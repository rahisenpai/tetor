## About tetor
tetor is a simple text editor developed in Python with Tkinter library for GUI and MySQL for back-end. It also supports standard keybindngs as shortcut to various options.

#### File Operations
- new (ctrl-n)
- open (ctrl-o)
- save (ctrl-s)
- save as (ctrl-s)

#### Editing
- cut (ctrl-x)
- copy (ctrl-c)
- paste (ctrl-v)
- undo (ctrl-z)
- redo (ctrl-y)
- change upper/lower case

#### View
- text wrap

#### Styling
- bold
- underline
- italics
- overstrike
- different fonts
- change text color
